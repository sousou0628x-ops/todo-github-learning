# タスク管理アプリ - GitHub学習プロジェクト

このプロジェクトは、GitHubの基本機能を学習するためのサンプルアプリケーションです。

## 現在のステータス

✅ **ステップ1: 基本的なGitワークフロー** - 完了  
✅ **ステップ2: ブランチ戦略** - 完了  
✅ **ステップ3: Issue/Pull Request** - 完了  
🚧 **ステップ4: GitHub Actions（CI/CD）** - 進行中

## プロジェクト概要

シンプルなToDoリストアプリケーションを通じて、以下のGitHub機能を段階的に学習します：

1. ✅ 基本的なGitワークフロー（現在のステップ）
2. ⏳ GitHub Actions（CI/CD）
3. ⏳ Issue/Pull Requestテンプレート
4. ⏳ README、ライセンス、.gitignore

## 機能

- タスクの追加
- タスクの完了/未完了切り替え
- タスクの削除
- ローカルストレージでの永続化

## 使用方法

1. `index.html` をブラウザで開く
2. テキストボックスにタスクを入力
3. 「追加」ボタンをクリック
4. チェックボックスで完了/未完了を切り替え
5. 「削除」ボタンでタスクを削除

## 技術スタック

- HTML5
- CSS3
- JavaScript (Vanilla)
